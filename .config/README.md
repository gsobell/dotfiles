# dotfiles
A backup of my personal config files. Includes questionable, poorly implemented workarounds, use at your own risk.<sup>1</sup>. Attempting to conform to [XDG](https://specifications.freedesktop.org/basedir-spec/basedir-spec-latest.html) [Base Directory](https://wiki.archlinux.org/title/XDG_Base_Directory) Specifications, with the notable exception of `.vim` and `.moc` for convenience, and `.gtkrc-2.0` because of some evil unresolvable theming issues.

This README is placed here rather than the git root since I'm not a fan of a messy `$HOME` locally.

Remember, `pacman -Syu` early and `pacman -Syu` often.
